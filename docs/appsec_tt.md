title: Аббревиатурное описание AppSec инструментов
description: Описание аббревиатур, типов, классов для AppSec инструментов.
---

<table>
  <thead>
    <tr>
      <th style="width:10%;">Аббревиатура</th>
      <th style="width:90%;">Описание</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><strong>DAST</strong></td>
      <td>Dynamic Application Security Testing – динамическое тестирование безопасности приложения, имитация реальных атак на приложение во время его работы.</td>
    </tr>
    <tr>
      <td><strong>SAST</strong></td>
      <td>Static Application Security Testing – статический анализ кода без его запуска на наличие ошибок и уязвимостей в исходном коде.</td>
    </tr>
    <tr>
      <td><strong>SCA</strong></td>
      <td>Software Composition Analysis – анализ состава компонент (зависимостей) программного обеспечения.</td>
    </tr>
    <tr>
      <td><strong>OSA</strong></td>
      <td>Open Source Analysis – анализ компонентов с открытым исходным кодом при попадании в периметр разработки, включая безопасность, качество и лицензионные риски.</td>
    </tr>
    <tr>
      <td><strong>SCS</strong></td>
      <td>Secure Code Standards – набор правил и практик безопасной разработки, снижающих риск уязвимостей и утечек данных.</td>
    </tr>
    <tr>
      <td><strong>License Policy</strong></td>
      <td>Политики и проверки на соблюдение лицензионных соглашений и авторских прав при использовании сторонних компонентов.</td>
    </tr>
    <tr>
      <td><strong>SBOM</strong></td>
      <td>Software Bill of Materials – инвентаризационный список всех компонентов (пакетов, библиотек), используемых в приложении или необходимых для его работы.</td>
    </tr>
    <tr>
      <td><strong>NVS</strong></td>
      <td>Network Vulnerability Scanner – сетевой сканер уязвимостей с фокусом на уровнях L3/L4 и ограниченными DAST-возможностями.</td>
    </tr>
    <tr>
      <td><strong>BCA</strong></td>
      <td>Bytecode and Container Analysis – анализ бинарного кода и состава контейнеров.</td>
    </tr>
    <tr>
      <td><strong>CIS</strong></td>
      <td>Container Image Scanner – сканирование образов контейнеров на уязвимости (Container Security).</td>
    </tr>
    <tr>
      <td><strong>CSPM</strong></td>
      <td>Cloud Security Posture Management – контроль и аудит конфигураций Kubernetes и облачных сред.</td>
    </tr>
    <tr>
      <td><strong>SM</strong></td>
      <td>Secret Management – управление секретами (пароли, токены, ключи и т.п.).</td>
    </tr>
  </tbody>
</table>
